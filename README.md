# Employee Management System (CRUD)

This project is a basic CRUD (Create, Read, Update, Delete) application developed for the **NexTech Internship Assessment**. It allows managing employee records with image upload functionality and validation.

##  Features
- **Create:** Add new employees with Name, Email (Unique), Phone, Description, and Image.
- **Read:** View a list of employees with their profile images.
- **Update:** Edit employee details and replace profile images (automatically deletes old images to save space).
- **Delete:** Remove employee records and their associated images from the server.
- **Validation:** Strong server-side validation for required fields and unique emails.

##  Technology Stack
- **Framework:** Laravel 11
- **Language:** PHP 8.2+
- **Database:** MySQL
- **Frontend:** Blade Templates, Bootstrap 5
